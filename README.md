# Assets

Ce dépôt contient les ressources utilisées pour [ce projet VR](https://github.com/BlueBerry1589/Reincarnated-sorceror/).

Il inclut :
- le modèle de la mascotte ;
- le modèle du marteau ;
- les fichier blends pour les modèles ;
- une animation faite à la main (electrisation) et des animations provenant de Mixamo ;
- l'icône du jeu ;
- la feuille utilisée pour les instructions ainsi que son fichier de projet.
